# References and Example Projects

- Cursor: product-level hybrid search UX (exact + semantic).
- Obsidian: file-based Markdown knowledge base with graph view.
- deepset Haystack: open-source retrieval pipelines, connectors to vector stores.
- LlamaIndex / LangChain: indexing strategies and LLM connectors.
- Weaviate, Qdrant, Pinecone: vector databases with examples/demos.
- FAISS, Annoy: ANN libraries for large-scale nearest neighbor search.
- HDBSCAN / scikit-learn: clustering algorithms for embeddings.

Use these for architecture patterns, ingestion examples, and clustering approaches.
